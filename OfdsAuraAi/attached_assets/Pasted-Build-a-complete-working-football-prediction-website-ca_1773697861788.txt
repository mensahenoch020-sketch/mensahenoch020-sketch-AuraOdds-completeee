HOW PREDICTIONS WORK:
When page loads automatically predict ALL today's fixtures.
For each match:
1. Query matches table for H2H history between the two teams (last 20 meetings)
2. Query recent form for home team (last 10 matches)
3. Query recent form for away team (last 10 matches)
4. Calculate: home win %, away win %, draw %, average goals per game from H2H
5. Fetch live odds from Odds API
6. Pick best market based on stats:
   - home win prob > 55% → Home Win, Asian Handicap Home, 1UP Home, 2UP Home
   - away win prob > 50% → Away Win, Asian Handicap Away, 1UP Away, 2UP Away
   - draw prob > 35% → Draw, Double Chance 1X, Double Chance X2
   - avg goals > 2.8 → Over 2.5 Goals or BTTS Yes
   - avg goals < 2.0 → Under 2.5 Goals or BTTS No
7. Calculate Kelly Criterion: f = (b*p - q) / b, cap at 5%
8. Call Gemini AI to generate short sharp prediction (3 bullet points max)
9. Save prediction to picks table
10. Show on screen immediately
Predictions auto-refresh every 30 minutes.
Never show Kelly formula — just show the stake % result.

EXPRESS BACKEND ROUTES:

GET /api/fixtures
- Query matches table for fixtures where ft_home IS NULL and date between today and today+7
- Also call Football-Data.org competitions 2021, 2014, 2002, 2019, 2015, 2001
- Fetch odds from Odds API
- Return: { by_date: { "2026-03-17": [ { id, home, away, league, time, home_odds, draw_odds, away_odds } ] } }
- Cache in fixtures_cache table

GET /api/fixtures/today
- Same but only today's matches

POST /api/predict
- Body: { home, away, league, date, home_odds, draw_odds, away_odds }
- Fetch H2H and form from matches table
- Pick market, calculate edge and Kelly stake
- Call Gemini then Groq fallback
- Save to picks table
- Return: { pick, market, odds, confidence, edge, stake, analysis }

POST /api/accumulator
- Body: { matches: [{home, away, league, home_odds, draw_odds, away_odds}] }
- Predict each match
- Return: { analysis, combined_odds, picks: [{match, pick, odds, confidence}] }

GET /api/picks/today
GET /api/picks/all
